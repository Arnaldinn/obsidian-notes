O **`smbclient`** é uma ferramenta no **Linux** usada para acessar **compartilhamentos SMB (Server Message Block)** em redes Windows. Ele funciona como um **cliente SMB**, permitindo listar arquivos, transferir dados e interagir com servidores SMB.
```
smbclient -L //<IP_DO_SERVIDOR> -U <USUARIO>
```

O **`psexec.py`** é um script da **Impacket** que permite **execução remota de comandos** em sistemas Windows via **SMB (Server Message Block)**. Ele é amplamente usado em **pentests**, especialmente para ganhar **acesso remoto a máquinas Windows** após comprometer credenciais.

```
psexec.py ORIONSCORP2/rlourdes:senha123@192.168.1.10
```

O psexec.py esta nesse diretorio:

```
python3 /usr/share/doc/python3-impacket/examples/psexec.py orionscorp2/rlourdes:'georgeorwell1984'@172.16.1.253
```
